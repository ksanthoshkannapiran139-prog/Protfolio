# Santhosh K - Portfolio

A modern, responsive portfolio website for a Full Stack Developer and AIML Student. Built with HTML, CSS, and JavaScript, featuring smooth scrolling, reveal animations, and a typing effect.

## Features

- **Responsive Design**: Adapts to different screen sizes (desktop, tablet, mobile).
- **Smooth Scrolling**: Elegant transitions between sections.
- **Reveal Animations**: Elements fade in and slide up as you scroll.
- **Typing Effect**: Professional typing animation for the hero section.
- **Clean UI**: Minimalist design with a dark blue theme and cyan accents.

## Tech Stack

- **HTML5**: Structure and content.
- **CSS3**: Styling and animations.
- **JavaScript (Vanilla)**: Interactivity and animations.

## Getting Started

No installation required! Just open the `index.html` file in your web browser.
